# Seattle Airbnb Analysis

## Overview

## Goal
What affects pricing of airbnbs in Seattle? How do neighborhoods, time of year, and reviews affect overall price trends?

## Data Source and information
- This dataset, *Seattle Airbnb Open Data - SQL Project* was sourced from Kaggle and was created by Sharmaine Wong
- This dataset contains 3 CSV files, *calendar.csv* (7 columns, 2350878 rows), *listings.csv* (18 columns, 6442 rows), and *reviews.csv* (6 columns, 481350 rows)
- The dataset focuses on airbnb information in Seattle including dates, pricing, availability, listings, location, host, and reviews organized in 3 CSV files.
